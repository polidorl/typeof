Tipo de datos de una variable en JavaScript

Cómo detectar el tipo de datos de una variable en JavaScript

El operador typeof permite detectar el tipo de datos de una variable en JavaScript. Sin embargo, es importante tener en cuenta dos casos especiales: null y los arrays. El operador typeof devuelve "object" para null, lo cual es un error conocido en JavaScript. Para verificar si una variable es específicamente null, se debe realizar una comparación directa:variable === null; El operador typeof también devuelve "object" para los arrays, por lo que no puede distinguir entre un objeto y un array. Para verificar si una variable es un array, se utiliza el método Array.isArray(variable).

Recomendaciones:

  • Usa typeof para tipos primitivos como number, string,boolean, undefined y function.
  • Usa el método Object.prototype.toString para tipos más complejos como arrays,fechas u otros objetos.
  • Usa el método Array.isArray para verificar si una variable es un array.
  • Compara directamente con null o undefined para detectar esos valores específicos.

Utilizando una funcion:

                
                   // Definición de variables
                   let nombre = "Lisbeth";
                   let edad = 30;
                   let esEstudiante = true;
                   let datos = null;
                   let simbolo = Symbol('id');
                   let objeto = { nombre: "Lisbeth", edad: 30 };
                   let array = [1, 2, 3];
                   let funcion = function() { console.log("Hola"); };
                   
                   // Función para mostrar el tipo de dato
                   function mostrarTipo(variable, nombreVariable) {
                       let tipo = typeof variable;
                       // Caso especial para null y arrays
                       if (variable === null) {
                           console.log(`El tipo de dato de "${nombreVariable}" es: null`);
                               } else if (Array.isArray(variable)) {
                           console.log(`El tipo de dato de "${nombreVariable}" es: array`);
                       } else {
                           console.log(`El tipo de dato de "${nombreVariable}" es: ${tipo}`);
                       }
                   }

Llamadas a la función mostrarTipo:

                
                mostrarTipo(nombre, "nombre"); // string
                mostrarTipo(edad, "edad"); // number
                mostrarTipo(esEstudiante, "esEstudiante"); // boolean
                mostrarTipo(datos, "datos"); // null
                mostrarTipo(simbolo, "simbolo"); // symbol
                mostrarTipo(objeto, "objeto"); // object
                mostrarTipo(array, "array"); // array
                mostrarTipo(funcion, "funcion"); // function
